HTML5 IN SHORT

HTML is markup language for describing web documents.

HTML stands for Hyper Text Markup Language

HTML documents are described by HTML tags.

A sample HTML page

The above mentioned is the basic structure of a html page.
The first line is the HTML5 declaration. This conveys which version of HTML your document is using and a critical information for some processing tools.

<html>-- start tag </html>-- end tag. <title>-- This tag is used for defining the title of document . <head>-- This tag is used for declaring all the related script, css files used for building the document. <body>-- This tag is the main tag which describes the visible data content. All the tags should be always enclosed between <html></html> tags.

HTML TAGS:

HTML tags mostly come in pairs.
The first tag is start tag and second one is called end tag.
HTML tags are keywords surrounded by angular brackets.

<tagname>content</tagname>

Some widely used tags:

    • HTML headings:

      <h1>This is heading</h1> <h2>This is heading</h2> <h3>This is heading</h3> <h4>This is heading</h4> <h5>This is heading</h5> <h6>This is heading</h6> <h1> indicates most important heading <h6> indicates least important heading

      HTML paragraphs

      <p>This is paragraph</p> <p>This is another paragraph</p>

      HTML Links:

      <p>HTML links are defined by <a> tags.</p> <p><a href="www.google.com">Click here for Google </a></p> <p>Clicking on this will redirect to google home page.</p>

      HTML Images:

      <p>HTML images are defined by <img> tags.</p> <p><img src=”” alt=”My home” width=”100px” height=”100px”> ‘src’ indicates path of the image . ‘alt’ indicates text to be shown when image is not loaded properly. width , height provides dimensions of the image. </p> <p>Clicking on this will redirect to google home page.</p>
